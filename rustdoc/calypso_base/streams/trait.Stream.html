Expand description

A trait for streamed data. For a stream, EOF means either the end of file / input if this stream is backed by a file or string, or the end of the stream.

fn gorge_while(
    &mut self,
    func: impl FnMut(&<Self as Iterator>::Item, usize) -> bool
) -> usize

Keep consuming elements while the condition is true and there are remaining elements. Returns the number of elements consumed.

If func returns true for the element, consume it. Repeat until func returns false for an element.

func takes in the element first, then the number of elements consumed so far.
