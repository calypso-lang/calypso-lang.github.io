Generate errors or report synchronized errors.

Panic errors are handled with Results. They can be generated as follows:

ⓘ
gen_error!(Err(self => {
    E0005, fmt_values = "values";
    labels: [
        LabelStyle::Primary =>
            (source_id, span);
            "label message",
        LabelStyle::Secondary =>
            (source_id2, span2);
            "label message 2"
    ],
    notes: [
        "note 1",
        "note 2"
    ]
}) as ())?

The as () at the end is to change the Ok type of the result, if necessary. Panic errors should only be used if there is ABSOLUTE CONFIDENCE that there is NO way to recover from this error.
